### Start project
- create workshop dir & pull repos 
  ```bash
  mkdir ~/workshop/
  
  cd ~/workshop/
  
  git clone https://github.com/NikolaRusakov/swapi-gql-rest.git server
  
  git clone https://github.com/NikolaRusakov/swapi-workshop.git client

  ```

- start server
  ```bash
  cd server/
   
  npm i
  
  npm start
  ```

---

**Open New Terminal**

---
- start client
  ```bash
  webstorm ~/workshop/client/ 
  
  cd ~/workshop/client/
  
  npm i
  
  npm start
  ```
