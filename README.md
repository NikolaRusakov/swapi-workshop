### Start project

- create workshop dir & pull repos 
  ```bash
  mkdir ~/workshop/
  
  cd ~/workshop/
  git clone https://github.com/NikolaRusakov/swapi-workshop.git swapi

- install & start server/client

  ```bash
  webstorm ~/workshop/swapi/ 
  
  cd ~/workshop/swapi/
  
  npm run install:deps
  
  npm start
  ```
